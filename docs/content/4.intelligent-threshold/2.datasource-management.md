在“监控数据源管理”页面，您可以对平台的监控数据源进行统一管理。这些数据源主要服务于平台的各项智能监控功能。本文档将引导您完成相关操作。

## 连接管理

`连接管理`用于集中配置和维护所有外部监控系统的访问凭证。

![数据源管理页面](/veaiops/images/datasource-connection.png)

### 操作步骤

1.  在顶部导航栏点击 `系统配置`，进入 `监控数据源管理` 页面。
2.  点击页面右上角的 `管理连接` 按钮，即可打开连接管理弹窗。
3.  在弹窗中，您可以选择对应的监控系统标签页（如 `火山引擎`、`阿里云` 或 `Zabbix`）。
4.  点击 `+ 新建连接` 并填写连接参数。

### 配置连接

在`连接管理`页面，点击`新建连接`后，您需要根据所选的`连接类型`填写相应的连接参数。


::alert{type="info"}
#title
安全提示
#description
为了保障您的凭证安全，所有敏感信息（如 `Secret Access Key`、`AccessKey Secret` 及 `API Password` 等）在保存时均会经过加密处理。
::

::tabs
  ::tab{label="火山引擎"}
    ![火山引擎新建连接](/veaiops/images/datasource-volcengine-connection.png)
    - **`连接名称`**: 为此连接指定一个唯一的、有意义的名称。
    - **`AccessKey ID`**: 您的火山引擎账号的 AccessKey ID。
    - **`SecretAccess Key`**: 您的火山引擎账号的 SecretAccess Key。
  ::
  ::tab{label="阿里云"}
    ![阿里云新建连接](/veaiops/images/datasource-aliyun-connection.png)
    - **`连接名称`**: 为此连接指定一个唯一的、有意义的名称。
    - **`AccessKey ID`**: 您的阿里云账号的 AccessKey ID。
    - **`AccessKey Secret`**: 您的阿里云账号的 AccessKey Secret。
  ::
  ::tab{label="Zabbix"}
    ![Zabbix新建连接](/veaiops/images/datasource-zabbix-connection.png)
    - **`连接名称`**: 为此连接指定一个唯一的、有意义的名称。
    - **`API URL`**: Zabbix API 的访问地址。
    - **`API User`**: 用于 API 访问的用户名。
    - **`API Password`**: 对应的 API 密码。
  ::
::

::tip
未来，我们计划支持更多第三方监控系统，以支持更丰富的监控场景，敬请期待。
::

## 数据源管理

`数据源管理`是配置和管理所有外部监控系统数据源的中心。

![数据源管理页面](/veaiops/images/datasource-instance.png)

### 操作步骤

1.  在 `监控数据源管理` 页面，点击页面右上角的 `+ 新增数据源` 按钮。
2.  这将启动一个配置向导，引导您完成后续的配置流程。

### 配置向导

不同数据源类型的配置向导详情如下：

::tabs
  ::tab{label="火山引擎"}
    配置火山引擎数据源时，您需要依次选择连接、产品、子空间、监控项和实例。向导将引导您完成整个过程。

    ![火山引擎数据源配置向导](/veaiops/images/datasource-volcengine-wizard.png)
  ::
  ::tab{label="阿里云"}
    配置阿里云数据源时，您需要依次选择连接、命名空间、监控项和实例。向导将引导您完成整个过程。

    ![阿里云数据源配置向导](/veaiops/images/datasource-aliyun-wizard.png)
  ::
  ::tab{label="Zabbix"}
    配置 Zabbix 数据源时，您需要依次选择连接、模板、监控项和主机。向导将引导您完成整个过程。

    ![Zabbix数据源配置向导](/veaiops/images/datasource-zabbix-wizard.png)
  ::
::
