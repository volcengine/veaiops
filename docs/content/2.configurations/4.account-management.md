在“账号管理”页面，您可以对平台的用户账号进行统一管理。本文档将引导您完成相关操作。

平台内预设了两种用户角色，他们的权限分别如下：

::card-grid
  ::card{icon="i-lucide-user-cog" title="管理员"}
  拥有完整的用户管理权限，可以创建、查看和删除系统中的所有用户账号。
  ::
  ::card{icon="i-lucide-user" title="普通用户"}
  仅能查看用户列表，无法进行创建和删除操作。
  ::
::card-grid

::tip
未来，我们计划集成更多第三方用户认证体系，以支持更丰富的账号管理场景。
::


![账号管理页面](/veaiops/images/account-management-user-list.png)

## 操作指南


::tabs
  ::tab{label="新增用户"}
    1.  在“账号管理”页面，点击右上角的`+ 新增账号`按钮。
    2.  在弹出的窗口中，填写以下信息：
        - **用户名**: 用于登录的唯一名称。
        - **邮箱**: 用户的联系邮箱。
        - **密码**: 为用户设置的初始登录密码。
        - **角色**: 分配“管理员”或“普通用户”角色。
    3.  点击`创建`按钮，即可完成新账号的添加。

    ![新增账号](/veaiops/images/account-management-add-user.png)
  ::
  ::tab{label="删除用户"}
    1.  在用户列表中，找到需要删除的用户。
    2.  点击其对应的`删除`按钮。
    3.  在弹出的确认框中点击`确认`以完成删除。

    ![删除用户](/veaiops/images/account-management-user-list.png)

    ::tip{type="danger"}
      **注意**: 删除用户是不可逆操作，请谨慎处理。
    ::
  ::
  ::tab{label="修改密码"}
    1.  点击页面右上角的用户头像，从下拉菜单中选择`修改密码`。
    2.  在弹出的窗口中，输入您的“旧密码”、“新密码”并再次确认“新密码”。
    3.  点击`更新`按钮，完成密码修改。

    ![修改密码](/veaiops/images/account-management-edit-user.png)
  ::
  ::tab{label="退出登录"}
    1.  点击页面右上角的用户头像。
    2.  在下拉菜单中选择`退出登录`，即可安全退出当前账号。

    ![退出登录](/veaiops/images/account-management-logout.png)
  ::
::
