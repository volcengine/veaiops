在“事件中心” - “事件订阅”页面，您可以管理所有事件订阅规则。本文档将引导您完成相关操作。
“事件订阅”功能主要用于管理事件订阅，管理员用户可以查询、编辑和新增，普通用户仅支持查询。
根据智能体的不同，您可以通过以下入口进行访问：
- **智能阈值 | 事件订阅**: 在该模块可查看订阅了“智能阈值Agent”智能体的所有订阅规则。
- **ChatOps | 事件订阅**: 在该模块可查看订阅了“内容识别Agent”智能体的所有订阅规则。
- **事件中心 | 事件订阅**: 在该模块可查看所有的订阅规则。

## 操作指南
::tabs
  ::tab{label="浏览和筛选事件订阅"}
    事件订阅列表页面展示了所有事件订阅规则，并提供了筛选功能。
    ![事件订阅规则新建和编辑](/veaiops/images/configurations/subscribe-list.png)
    ### 筛选与重置
    您可以使用页面顶部的筛选器来快速查找您关心的事件订阅规则：
    - **名称**: 根据事件订阅规则名称进行模糊匹配筛选。
    - **智能体**: 根据选择的智能体进行精准筛选。
    - **事件级别**: 根据事件级别进行精准筛选。
    - **是否开启WEBHOOK**: 根据是否开启WEBHOOK精准筛选。
    - **关注项目**: 根据关注项目精准筛选。
    完成筛选条件设置后，点击`重置`按钮可以清除所有筛选条件，恢复查询所有事件订阅规则。
  ::
  ::tab{label="新建和编辑事件订阅"}
    点击页面上角的`+新建订阅`按钮，可以打开侧边栏，新建事件订阅规则。
    点击事件订阅列表中任一条目右侧的`编辑`按钮，可以打开侧边栏，编辑改事件订阅规则。
    ![策略新建和编辑](/veaiops/images/configurations/subscribe-edit.png)
    ### 编辑页面说明
    - **名称**: 标识事件订阅的名称，不允许为空，支持列表页模糊搜索。
    - **智能体**: 订阅的智能体，每个订阅规则必须关联一个智能体，不能支持为空。（如果想要订阅多个智能体，需要创建多个事件订阅规则）
    - **生效时间**: 订阅规则生效时间，不允许为空，默认开始时间为创建记录时的时间，默认结束时间为创建记录时的时间往后推100年。
    - **关注项目**: 系统支持订阅不同关注项目的事件，当不选择任何项目时，会订阅所有项目属性（包括无项目属性)的事件。
    - **订阅的事件级别**: 事件订阅规则订阅的事件级别，此项为必选项，可以订阅多个级别的事件。
    - **消息卡片通知策略**: 符合订阅规则的事件可以通过关联的消息卡片通知策略分发的对应的群，如果不关联任何消息卡片通知策略，则命中该订阅规则的事件不会发送任何消息卡片。
    - **WEBHOOK 投递事件**: 当需要通过WEBHOOK通知事件时，您可以打开**开启 WEBHOOK 投递**开关，并配置WEBHOOK地址作为投递目标，开启WEBHOOK投递后，该字段必填，同时您可以配置扩展请求头，用于自定义请求头信息，例如认证信息等。
    ::tip
      WEBHOOK 配置提示
      - 您需要确保 WEBHOOK 地址能够在 VeAIOps 部署所在集群的网络环境中可正常访问，否则系统将无法发送告警事件到该地址。
    ::

  ::
  ::tab{label="删除事件订阅"}
    1. 在列表页，找到需要删除的事件订阅规则。
    2. 点击其对应的 `删除` 按钮。
    3. 在弹出的确认框中点击 `确认` 以完成删除。
    ::alert{type="danger"}
    **注意**: 删除事件订阅是不可逆操作，请谨慎处理。
    ::
  ::

::
