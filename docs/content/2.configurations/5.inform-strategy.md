在“事件中心” - “消息卡片通知策略”页面，您可以管理所有消息卡片通知策略。本文档将引导您完成相关操作。
“消息卡片通知策略”功能主要用于管理消息卡片通知策略，管理员用户可以查询、编辑和新增，普通用户仅支持查询。
## 操作指南
::tabs
  ::tab{label="浏览和筛选消息卡片通知策略"}
    消息卡片通知策略列表页面展示了所有消息卡片通知策略，并提供了筛选功能。
    ![策略新建和编辑](/veaiops/images/configurations/inform-strategy-list.png)
    ### 筛选与重置
    您可以使用页面顶部的筛选器来快速查找您关心的消息卡片通知策略：
    - **策略名称**: 根据消息卡片策略名称进行模糊匹配筛选。
    - **企业协同工具**: 根据企业协同工具进行筛选，当前仅支持飞书。
    - **机器人**: 根据推送机器人进行筛选，当前仅支持飞书机器人。
    完成筛选条件设置后，点击`重置`按钮可以清除所有筛选条件，恢复查询所有消息卡片通知策略。
  ::
  ::tab{label="新建和编辑消息卡片通知策略"}
    点击页面上角的`+新建策略`按钮，可以打开侧边栏，新建消息卡片通知策略。
    点击列表中任意消息卡片通知策略的`编辑`按钮，可以打开侧边栏，编辑该消息卡片通知策略。
    ![策略新建和编辑](/veaiops/images/configurations/inform-strategy-edit.png)
    ### 编辑页面说明
    - **策略名称**: 标识消息卡片通知策略的名称，不允许为空，支持列表页模糊搜索。
    - **描述**: 记录消息卡片通知策略的描述信息，支持为空，不支持检索。
    - **企业协同工具**: 承载消息卡片通知机器人的企业协同工具，目前仅支持飞书。
    - **机器人**: 承载消息卡片通知的机器人，选定企业系统工具后，可下拉选择已经创建好的机器人。 您可以在“系统配置” - “群聊机器人管理” 页面管理。
    - **通知群**: 当事件订阅规则关联了此消息卡片通知策略后，事件可以被分发通知的群列表，您可以选择一个或多个群，但至少选择一个群。
  ::
  ::tab{label="删除消息卡片通知策略"}
    1. 在列表页，找到需要删除的通知策略。
    2. 点击其对应的 `删除` 按钮。
    3. 在弹出的确认框中点击 `确认` 以完成删除。

    ::alert{type="danger"}
    **注意**: 删除通知策略是不可逆操作，请谨慎处理。
    ::
  ::
::

::tip{type="danger"}
**注意**: 启用消息卡片通知策略前，您需要先添加群聊机器人，创建群聊并邀请机器人到期望被通知的群聊中，同时必须在本系统中根据引导设置卡片模版。
::
