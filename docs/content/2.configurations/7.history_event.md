在“历史事件”页面，您可以查看和检索所有由平台记录的历史事件。本文档将引导您完成相关操作。

“历史事件”功能主要用于记录由各类智能体（如“内容识别Agent”、“主动回复Agent”、“被动回复Agent”、“智能阈值Agent”等）推送的所有事件。这对于事后追溯、审计或问题排查非常有帮助。

根据事件来源的不同，您可以通过以下入口进行访问：
- **事件中心**: 在该模块可查看所有历史事件。
- **智能阈值**: 在该模块可查看由“智能阈值Agent”产生的历史事件。
- **ChatOps**: 在该模块可查看由“内容识别Agent”、“主动回复Agent”和“被动回复Agent”产生的历史事件。

![历史事件页面](/veaiops/images/history-event-list.png)

## 操作指南

::tabs
  ::tab{label="浏览和筛选事件"}
    事件列表页面展示了所有历史事件的核心信息，并提供了强大的筛选功能。

    ![历史事件页面](/veaiops/images/history-event-list.png)

    ### 筛选与重置

    您可以使用页面顶部的筛选器来快速定位特定事件：
    - **智能体**: 根据产生事件的智能体进行筛选。
    - **事件级别**: 根据事件的级别进行筛选。
    - **状态**: 根据事件的状态进行筛选。

    完成筛选条件设置后，点击`重置`按钮可以清除所有筛选条件，恢复显示所有事件。
  ::
  ::tab{label="查看事件详情"}
    点击任意事件记录右侧的`查看详情`按钮，可以打开侧边栏，查看该事件的完整信息。

    ![事件详情](/veaiops/images/history-event-raw-data.png)

    ### 详情板块说明

    - **基础信息**: 展示事件的基本属性，如智能体、事件状态、事件级别、事件ID和项目信息。
    - **时间信息**: 包括事件的精确创建和更新时间。
    - **渠道信息**: 如果事件被推送，这里会显示推送渠道的详情。
    - **原始数据**:
      - 提供事件的完整原始数据，支持`格式化视图`和`JSON视图`两种模式切换。
      - 您可以方便地`复制`数据内容，或将其`下载`为 JSON 文件进行本地分析。
