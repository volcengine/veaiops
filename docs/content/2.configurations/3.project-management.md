在“项目管理”页面，您可以对平台的项目进行统一管理。本文档将引导您完成相关操作。


在VeAIOps中，我们通过项目来分组和筛选各类配置项，如可通过项目来区分管理不同的智能阈值任务、事件等。在您的实际使用中，项目可以配置为具体的工作项目、组织结构、产品线等实际用于分类管理配置资源的实体。



![项目管理页面](/veaiops/images/project-management-ui.png)

## 操作指南

::tabs
  ::tab{label="新建项目"}
    点击页面右上角的`新建项目`按钮，可以创建一个新的项目。

    ![新建项目](/veaiops/images/project-management-new.png)

    ### 参数说明

    - **项目ID**: 项目的唯一标识，例如 `project-a`。
    - **项目名称**: 项目的显示名称，例如 `项目A`。
  ::
  ::tab{label="导入项目"}
    为了提高效率，平台支持通过文件批量导入项目。点击页面右上角的`导入项目`按钮，可以批量导入项目数据。

    ![导入项目](/veaiops/images/project-management-import.png)

    ### 导入步骤

    ::steps
    ### 准备数据文件
      根据指引准备一个 CSV 格式的数据文件。您可以点击`下载模版文件`来获取一个包含正确表头的模版。
      - **文件格式**: CSV (逗号分隔)。
      - **文件编码**: UTF-8。
      - **必填字段**: `project_id`, `project_name`。
    ### 上传文件:
    将准备好的 CSV 文件拖拽到上传区域，或点击该区域选择文件进行上传。
    ### 开始导入
    点击`开始导入`按钮，系统将开始处理文件并导入数据。
    ::

    ### 注意事项
    - 在导入过程中，请不要关闭或刷新浏览器页面。
    - 如果待导入文件中的某个 `project_id` 在系统中已经存在，系统将自动跳过该条记录，不会进行覆盖或更新。
    ::
  ::
::
