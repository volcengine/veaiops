在“卡片模版管理”页面，您可以对平台中用于机器人的卡片消息模版进行统一管理。这些模版定义了机器人在企业协同工具（如飞书）中发送的卡片消息的样式和内容，是实现丰富交互体验的关键。本文档将引导您完成相关操作。

![卡片模版管理页面](/veaiops/images/card-template-management-ui.png)

## 操作指南

::tabs
  ::tab{label="新建模版"}
    点击页面右上角的`新建模版`按钮，可以创建一个新的卡片模版。

    ![新建卡片模版](/veaiops/images/card-template-management-new.png)

    ### 飞书操作指引

    ::steps
    ### 下载卡片模版
      在“新建卡片模版”窗口中，点击`下载卡片模版`按钮，获取一个基础文件。
      ::tip
      请先下载我们为您准备的卡片文件，作为后续导入的基线。
      ::
    ### 导入卡片
      1. 登录您的企业协同工具官方提供的卡片搭建工具（例如 [飞书卡片搭建工具](https://open.feishu.cn/cardkit)）。
      2. **导入卡片模版**: 将下载的模版文件导入到卡片搭建工具中。
        ![导入卡片模版](/veaiops/images/card-template-management-lark-import.png)
        如果您在卡片搭建工具中从未创建过卡片模版，请先创建空白卡片后再导入：
        ![新建后导入卡片模版](/veaiops/images/card-template-management-lark-create-import.gif)
      3. **保存并发布卡片模版**: 在卡片搭建工具中保存并发布您的卡片模版，获取模版ID。
          1. 发布后添加所有应用:
          ![发布后添加所有应用](/veaiops/images/card-template-management-lark-publish.gif)
          2. 您也可以随时管理"自定义机器人/应用"，按需添加相关应用:
          ![自定义机器人/应用](/veaiops/images/card-template-management-lark-bot-auth.png)
      4. **配置卡片模版**: 回到本平台，将获取到的模版ID及相关信息填入“填写模版信息”表单中。
      5. **保存**: 点击`保存`按钮，即可完成模版的创建。
    ::

    ### 参数说明

    - **模版ID**: 在企业协同工具中发布的卡片模版ID (可以为不同智能体配置相同的模版ID)。
    - **智能体**: 选择将要使用此卡片模版的智能体。当该智能体需要发送消息时，会调用此模版来渲染内容。
    - **企业协同工具**: 选择该模版将应用在哪个企业协同工具平台。

  ::
  ::tab{label="编辑模版"}
    点击模版列表中任一项操作列的`编辑`按钮，可以修改该模版的配置信息。

    ![编辑卡片模版](/veaiops/images/card-template-management-edit.png)

    编辑流程与新建模版基本一致，您可以更新模版ID、关联的智能体等。

  ::
  ::tab{label="删除模版"}
    1. 在模版列表中，找到需要删除的模版。
    2. 点击其对应的`删除`按钮。
    3. 在弹出的确认框中点击`确认`以完成删除。

    ![卡片模版删除页面](/veaiops/images/card-template-management-ui.png)

    ::alert{type="danger"}
    **注意**: 删除模版是不可逆操作，请谨慎处理。
    ::
  ::
::
