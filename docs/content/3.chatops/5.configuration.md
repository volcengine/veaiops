::

## 操作指南

::steps
### LLM 配置

推荐使用火山方舟服务

![火山方舟在线推理](/images/volcengine-ark-model.png)

1. **模型名称** / **Embedding模型名称**：复制模型接入点（EP）作为模型名称
  ::tip
  请选用多模态模型，能更好支持图文理解
  建议使用 Doubao-Seed-1.6 & Doubao-embedding 模型
  ::
1. **API Base URL**：默认 ```https://ark.cn-beijing.volces.com/api/v3```
2. **API Key**：跳转[火山方舟API Key管理](https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey){:target="_blank" rel="noopener noreferrer"}页面获取

### 火山引擎知识库

当开启ChatOps能力后，智能体会自动创建知识库并存放于TOS平台，并自动导入到知识库中，故您需要填写如下配置信息，如果填写错误，知识库相关功能将不可用
1. **Access/Secret Key**：请跳转[访问控制-API访问密钥](https://console.volcengine.com/iam/keymanage){:target="_blank" rel="noopener noreferrer"}获取有效的Access Key/Secret Access Key。
  请确保该账号拥有以下权限：
  * TOSFullAccess：知识库原文档的存储，并通过TOS导入火山引擎知识库
  * VikingdbFullAccess：知识库 & 记忆库（使用前请[开通该功能](https://console.volcengine.com/vikingdb/region:vikingdb+cn-beijing/home?projectName=default)）
  * ArkFullAccess：方舟平台
   ![火山账号权限配置](/images/volcengine-access-management.png)
2. **TOS区域**：选择火山引擎TOS桶所在地域
3. **网络类型**：如果您的VeAIOps服务部署于火山引擎的资源，请选择内网，否则需要选择公网（可能会产生额外的公网流量和费用）；
4. **火山引擎知识库集合**：您可以输入上述主账号下的其他[知识库](https://console.volcengine.com/ark/region:ark+cn-beijing/knowledge/collection/list){:target="_blank" rel="noopener noreferrer"}作为备选，需要输入知识库ID，系统支持配置多个知识库
5. **查看自动创建的知识库/记忆库/TOS**:
   * 您可以在[火山知识库管理界面](https://console.volcengine.com/vikingdb/knowledge/region:vdb-knowledge+cn-beijing/collection/list?projectName=default){:target="_blank" rel="noopener noreferrer"}查看和管理这些知识库（知识库名称为`veaiops_autodoc/autoqa`）。
   * 您可以在[火山记忆库管理界面](https://console.volcengine.com/vikingdb/memory/region:vdb-memory+cn-beijing/list?projectName=default){:target="_blank" rel="noopener noreferrer"}查看和管理这些记忆库（记忆库名称为`veaiops_reactive_agent`）。
   * 您也可以在[火山TOS管理界面](https://console.volcengine.com/tos/bucket?projectName=default){:target="_blank" rel="noopener noreferrer"}查看和管理这些知识库（TOS名称为`veaiops-autodoc/autoqa`）。
   * (注：以上资源均注册在对应的火山引擎AK下，请确保您使用的是正确的火山引擎账号)
