**VeAIOps（Volcengine AIOps Suite）**，是由[火山引擎](https://www.volcengine.com/)
推出的为用户提供开箱即用的智能运维套件，支持智能应急协同、智能监控告警等核心功能场景。

## 核心能力

- **快速部署**：基于 Helm Chart，一键部署到 Kubernetes 集群
- **ChatOps**：通过智能化手段提升在群聊/Oncall（问题咨询、告警排查、事故处理等）场景下的问题感知能力与问题处理效率。
- **智能阈值**：基于机器学习的自适应阈值推荐，自动识别异常指标
- **灵活扩展**：基于 [VeADK](https://volcengine.github.io/veadk-python/) 开发，支持用户自定义扩展，可方便地支持更多数据源和运维场景

## 可观测性

- 兼容 OpenTelemetry 协议，支持指标、日志和链路追踪

## 部署架构

![部署架构](/veaiops/images/veaiops-deploy-structure.png)

### 核心组件

- **Backend**：后端服务，提供配置管理和事件中心
- **Frontend**：Web 管理界面
- **ChatOps**：智能运维 Agent 服务
- **Intelligent Threshold**：智能阈值算法服务
